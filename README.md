# Career Prediction Website Using Machine Learning

## 📌 Project Overview
The Career Prediction Website is a web-based application that helps students identify suitable career paths based on their skills, interests, and academic performance. The system uses Machine Learning to analyze user inputs and recommend the most appropriate career options.

## 🚀 Features
- User Registration and Login
- Secure Authentication
- Career Prediction using Machine Learning
- User-friendly Interface
- Input Validation
- Career Recommendation Results
- Admin Panel for Managing Users

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Deepthi20029/Career_Prediction_Website.git
```

### Navigate to the Project

```bash
cd Career_Prediction_Website
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run the Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
